# Setup Status Checklist

Track what you have configured. Update `[ ]` to `[x]` as you complete each item.

> For guided setup of any item, run `/setup-helper [feature-name]`.

## Personalization (claude.ai)

- [ ] Profile preferences set (Settings > Profile)
      `/setup-helper personalization`
- [ ] At least one custom style created
      `/setup-helper styles`
- [ ] Memory enabled (Settings > Capabilities)
      `/setup-helper memory`
- [ ] Memory summary reviewed (Settings > Capabilities > View and edit memory)

## Projects (claude.ai)

- [ ] At least one project created
      `/setup-helper projects`
- [ ] Project instructions written
- [ ] Project knowledge uploaded
- [ ] Project starred for quick access

## Integrations (claude.ai)

- [ ] GitHub connected
      `/setup-helper integrations`
- [ ] Google Drive connected
      `/setup-helper google-drive`
- [ ] Gmail connected
      `/setup-helper gmail-calendar`
- [ ] Google Calendar connected
- [ ] Connectors directory browsed (claude.ai/directory)
      `/setup-helper connectors`

## Claude Code -- Memory

- [ ] Project CLAUDE.md created (`/init`)
      `/setup-helper claude-md`
- [ ] User memory set (`~/.claude/CLAUDE.md`)
- [ ] Local memory created (`./CLAUDE.local.md`)
- [ ] Project rules created (`.claude/rules/`)

## Claude Code -- Styles & Skills

- [ ] Output style chosen (`/output-style`)
      `/setup-helper output-styles`
- [ ] Custom output style created (`.claude/output-styles/`)
- [ ] Custom skills created (`.claude/skills/`)
      `/setup-helper skills`

## Claude Code -- Extensions

- [ ] MCP servers configured (`.mcp.json`)
      `/setup-helper mcp`
- [ ] Subagents created (`.claude/agents/`)
      `/setup-helper agents`
- [ ] Hooks configured (`.claude/settings.json`)
      `/setup-helper hooks`
- [ ] Plugins explored (`/plugin`)
      `/setup-helper plugins`
- [ ] Settings reviewed (`/config`)
      `/setup-helper settings`
