# Anthropic Quick Reference

Company info, career development, and research resources.

## Constitutional AI (CAI)

Core principle: AI trained to be harmless, honest, honest, and helpful without human feedback at scale.

**Key Concepts:**
- Self-critiquing: Model evaluates its own outputs against principles
- Iterative improvement: Based on a "constitution" of values
- Reduced reliance on human raters: More scalable than RLHF alone

## Claude AI Assistant

**Latest Models:**
- Claude Opus 4.5 (most capable, ~200K context)
- Claude Sonnet 4.5 (balanced)
- Claude Haiku 4.5 (fast, cost-effective)

**Capabilities:**
- Text generation and analysis
- Code writing and debugging
- Document processing and summarization
- Tool use (function calling)
- Vision (image understanding)
- Extended thinking (reasoning)
