# Creating and Managing Projects in Claude

Claude lets you organize your work into projects for better collaboration and focus. Here’s how to use project features:

## 1. Creating Projects
- Start a new project from your Claude dashboard.
- Name your project and add a description to keep things organized.

## 2. Organizing Workspaces
- Group related chats, files, and tasks within a project.
- Use folders or tags to keep everything tidy.

## 3. Managing Project Members
- Invite team members to collaborate on a project.
- Assign roles and permissions to control access.
- Remove members or change roles as needed.

## 4. Project Settings
- Edit project details at any time.
- Archive or delete projects when they’re no longer needed.
