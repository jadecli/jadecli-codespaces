# Using the Google Drive Integration with Claude

Claude can connect to your Google Drive for seamless file access and management. Here’s how to use this integration:

## 1. Linking Google Drive
- Go to Claude’s integrations or settings page.
- Select Google Drive and follow the prompts to connect your account.
- Grant the necessary permissions for file access.

## 2. Accessing and Managing Files
- Browse, search, and open Google Drive files directly from Claude.
- Attach files from Drive to your chats or projects.
- Save new files or export content to your Drive.

## 3. Security and Permissions
- You control which files and folders Claude can access.
- Revoke access at any time from your Google account settings.
- All file access is secure and private.
