# Connect Your Tools to Claude

Enhance Claude’s capabilities by connecting your favorite tools and services. Here’s how to get started:

## 1. Supported Integrations
- Claude supports integrations with Google Drive, Gmail, Google Calendar, and more.
- Check the integrations page for the latest supported tools.

## 2. Connecting Tools
- Go to Claude’s integrations or settings section.
- Select the tool you want to connect and follow the prompts.
- Authorize access and set permissions as needed.

## 3. Managing Integrations
- View all connected tools in your integrations dashboard.
- Disconnect or update permissions at any time.

## 4. Unlocking New Features
- Connected tools enable smarter AI suggestions, file access, and workflow automation.
- Use integrations to streamline your daily tasks and boost productivity.
